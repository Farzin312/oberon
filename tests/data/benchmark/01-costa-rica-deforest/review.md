# 01 — Costa Rica Deforestation (Osa Peninsula)

## Location
Osa Peninsula, Puntarenas Province, Costa Rica. ~8.42N, -83.52W.

## Change description
Selective logging and smallholder clearing in tropical rainforest.
Visible canopy loss over a 6-month period (dry to wet season 2024).

## Source of truth
- Global Forest Watch tree cover loss alerts (GLAD)
- Hansen et al. Global Forest Change v1.11
- Visually confirmed in Sentinel-2 imagery

## Expected behavior
Pipeline should detect 1-5 findings with significant NDVI decrease (-0.15 to -0.50).
No abstention expected (tropical, but dry season start provides clear imagery).
