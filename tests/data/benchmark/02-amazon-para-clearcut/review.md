# 02 — Amazon Para Clearcut

## Location
Para State, Brazil. ~-7.47S, -55.17W. Southern Amazon deforestation arc.

## Change description
Industrial-scale clearcutting. Large rectangular patches of forest removal.
This area has sustained deforestation pressure from cattle ranching and soy.

## Source of truth
- PRODES (Brazilian National Institute for Space Research) annual deforestation
- Hansen Global Forest Change
- DETER alerts

## Expected behavior
Multiple findings expected (2-8), large NDVI drop (-0.20 to -0.60).
Clear geometry visible even in 10m Sentinel-2.
